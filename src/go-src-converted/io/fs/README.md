# go.io.fs

> C# package converted from the Go standard library by [go2cs](https://github.com/GridProtectionAlliance/go2cs).
> Go version: 1.23.1

Package fs defines basic interfaces to a file system. A file system can be provided by the host operating system but also by other packages.

See the [testing/fstest](/testing/fstest) package for support with testing implementations of file systems.
